N846 SUPPLEMENTAL CO-16

NDC Does Not Correspond to HCPCS/CPT Billed — Drug Billing / Data Mismatch

Drug Billing · Professional & Facility Claims
Common CARC Pairing
CO-16 CMS-1500 UB-04
What This Means

The National Drug Code (NDC) reported on the claim does not match the HCPCS/CPT drug code (typically a J-code) billed on the same service line. When this happens, payers flag it because the supplied drug and the billed code don't align — resulting in claim denial or suspension until clarified.

ℹ️
Important Distinction
N846 does not automatically mean the drug is not covered or the claim is permanently denied. It means the NDC and J-code need to match up before processing can continue.
Where to Report This Information
CMS-1500 (Professional)
  • Box 24D: HCPCS/CPT drug code (J-code)
  • Shaded area 24A–24G: N4 qualifier + 11-digit NDC + UOM + quantity
✅ Correct: N412345678901 UN1 ❌ Incorrect: N4 12345678901 UN1 (space after N4)

UB-04 (Facility)
  • FL44: HCPCS/CPT code (J-code)
  • FL43: N4 qualifier + 11-digit NDC + UOM + Quantity
Format: N4 + 11-digit NDC + UOM + Qty Example: N412345678901 UN1

⚠️
N4 Qualifier Format — Common Sub-Error
The N4 qualifier must appear as a single unspaced prefix directly before the 11-digit NDC. A space between N4 and the NDC number will cause a format rejection even if the NDC itself is correct.
Why This Happens
Data Entry & Selection Issues
  • Wrong NDC selected for the drug actually administered
  • NDC from a different manufacturer was used
  • NDC taken from the outer carton instead of the actual vial or syringe
  • A 10-digit NDC was not converted to the required 11-digit (5-4-2) format
  • The HCPCS/J-code does not align with the NDC submitted
  • NDC is expired, discontinued, or no longer active in the payer's drug file
Unit of Measure (UOM) Misalignment
UOM CodeWhen to Use
UNPowder-form drugs in a vial that must be reconstituted. Bill one unit per vial used.
MLLiquid-form drugs supplied in a vial. Bill actual milliliters administered.
GROintments, creams, inhalers, or bulk powder. Primarily retail pharmacy — rare in physician billing.
F2Primarily for Factor VIII Antihemophilic Factor billing.
How to Fix It
4a. Standard Correction Process
  1. 1Pull the MAR or vial label and confirm exactly what drug was administered
  2. 2Find the correct 11-digit NDC from the manufacturer of the drug actually given
  3. 3Convert any 10-digit NDC to 11-digit (5-4-2) format if needed
  4. 4Confirm the N4 qualifier is formatted correctly with no spaces
  5. 5Double-check UOM and quantity match what was documented in the MAR
  6. 6Validate the NDC aligns with the billed J-code using a current NDC-to-J-code crosswalk
  7. 7Submit a corrected claim with Resubmission Code 7
4b. Discontinued or Inactive NDC
  • Check the FDA NDC database or drug wholesaler records for a valid replacement NDC from the same manufacturer for the same drug
  • If the drug has been reformulated or the manufacturer has changed, identify the NDC that was active on the date of service
  • If no valid NDC can be confirmed for the DOS, escalate to coding — a NOC (not otherwise classified) code may be required with supporting documentation
Modifiers to Know
ModifierDescriptionUsage Notes
KOSingle drug unit dose formulationUse for single-drug unit-dose packaging.
KPFirst drug in a multiple drug unit dose formulationFirst drug when multiple drugs are in one unit-dose.
KQSecond or subsequent drug in a multiple drug unit dose formulationEach additional drug after KP in multi-drug unit-dose.
JWDrug amount discarded or not administered to the patientRequires a separate line on the same claim. Primary line must show >0 units administered. JW line shows wasted/discarded quantity. Both lines must reflect the same NDC and J-code.
JZZero wastage — no drug was discardedUse when no drug was discarded. Required by some payers when JW is not applicable.
💊
JW Modifier — Correct Billing Structure Example
Line 1: J-code | 2 ML administered | same NDC | same DOS
Line 2: J-code + JW | 0.5 ML discarded | same NDC | same DOS
If the JW line is the only line, or if the administered line shows zero units, the claim will deny.
Payer-Specific NDC Reporting Requirements

NDC reporting rules vary by payer. Verify requirements before submitting corrected claims:

Payer TypeKey Requirements
Medicare (Part B)NDC required on most drug claims via clearinghouse loop 2410. Format: N4 + 11-digit NDC + UOM + quantity. Refer to MAC LCD and CMS transmittals for drug-specific guidance.
MedicaidState-by-state variation. Many states require NDC on all drug lines including vaccines. Check state Medicaid billing manual.
Commercial / Managed CareRequirements vary by contract. Some payers require NDC in addition to J-code; others accept J-code only. Confirm payer-specific EDI companion guide.
Medicare AdvantageFollows Medicare Part B NDC rules in most cases but confirm with plan-specific requirements.
What NOT to Do
  • Do not appeal before correcting the NDC/J-code mismatch
  • Do not reuse old, default, or copied NDCs from previous claims without verification
  • Do not guess at the manufacturer, quantity, or drug lot
  • Do not submit a corrected claim with the same incorrect NDC
  • Do not appeal a N846 denial if you have not yet submitted a corrected claim first
🚫
Appealing before correcting the underlying NDC error wastes timely filing days and may forfeit your right to correct the claim. Always correct first, then appeal only if the corrected claim is also denied.
Prevention Checklist

Before submitting any drug claim, confirm:

  • ✅NDC captured directly from the administered vial or package label
  • ✅J-code validated against NDC using a current, reliable crosswalk
  • ✅10-digit NDC converted to 11-digit format (5-4-2) if applicable
  • ✅N4 qualifier formatted with no space before the NDC number
  • ✅Manufacturer-specific NDC confirmed (not from a different lot or supplier)
  • ✅UOM and quantity accurate per MAR documentation
  • ✅JW modifier applied on a separate line with administered quantity on primary line (if wastage applies)
  • ✅NDC confirmed as active on the date of service
  • ✅Claim run through edits before submission
Best Practice — When to Appeal vs. When to Correct

✅ Appeal Only When ALL of These Are True

  • A corrected claim has already been submitted and denied again
  • All billing data (NDC, J-code, UOM, quantity, manufacturer) has been confirmed correct against the MAR and vial label
  • The denial reason after correction still cites N846 — indicating a payer-side processing error

🔄 Correct & Resubmit First (Most Cases)

  • Most N846 denials resolve by correcting and resubmitting the claim
  • Do not appeal as a first step
  • Verify NDC ↔ J-code crosswalk, 11-digit format, N4 qualifier, UOM, and quantity before resubmission
  • Use Resubmission Code 7 on the corrected claim

Appeal Process & Documentation
⚠️
Coding Escalation — When to Escalate
Escalate when: multiple NDCs could legitimately map to the same J-code; drug wastage/JW is involved; compound drugs are billed; NOC codes are used; NDC is discontinued; or documentation conflicts between MAR, vial label, and charge entry.
Mode of Appeal
⚠️
Fax — Attention To Required
Always include "Attention To:" on the cover sheet when submitting by fax.
Required Documents Checklist
  • Denied EOB / ERA — Always required (mandatory, no exceptions)
  • MAR (Medication Administration Record) from date of service
  • Vial or package label confirming administered drug & NDC
  • Corrected Claim with verified NDC in proper 11-digit format
  • NDC ↔ J-code crosswalk / coding validation documentation
  • Drug Invoice (if applicable)
  • Appeal Cover Letter
